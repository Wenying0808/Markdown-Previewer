Markdown Guide
How to create Headings?
To create a heading, add number signs (#) in front of a word or phrase. The number of number signs you use should correspond to the heading level.

# Heading level 1
## Heading level 2
### Heading level 3
How to create Paragraph?
To create paragraphs, use a blank line to separate one or more lines of text.
How to create Emphasis?
To bold text, add two asterisks (**) or underscores (__) before and after a word or phrase.

**bold text**

To italicize text, add one asterisk (*) or underscore(_) before and after a word or phrase.

*italicized text*
How to create Lists?
To create an unordered list, add dashes (-), asterisks (*), or plus signs (+) in front of line items. Indent one or more items to create a nested list.
How to create Blockquotes?
To create a blockquote, add a > in front of a paragraph.

> This is blockquote.
How to create Code?
To denote a word or phrase as code, enclose it in backticks (`). If the word or phrase you want to denote as code includes one or more backticks, you can escape it by enclosing the word or phrase in double backticks (``).
How to create Links?
To create a link, enclose the link text in brackets [Name of link] and then follow it immediately with the URL in parentheses (url).

My favorite search engine is [Duck Duck Go](https://duckduckgo.com)
